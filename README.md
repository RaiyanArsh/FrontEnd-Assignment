# Getting Started
1. Clone the Repository
```
git clone https://github.com/RaiyanArsh/FrontEnd-Assignment.git
```
2. Change directory to main directory
```
cd FrontEnd-Assignment
```
3. Install dependencies
```
npm install
```
4. Run the Project
```
npm run dev
```
